# Changelog

All notable changes to the "Nyxora" extension will be documented in this file.

## [1.0.1] - 2026-07-22 — Patch Release

### Changed
- Updated extension branding, display name, and package name to **Nyxora**.
- Updated high-resolution extension icon (`icon.png`).
- Refined documentation and marketplace installation steps.

## [1.0.0] - 2026-07-22 — Initial Release

### Added
- **Nyxora Midnight** — flagship dark theme with deep blue-black tones
- **Nyxora Aurora Light** — light theme with soft matcha/ivory aesthetics
- **Nyxora Aurora Dark** — dark counterpart to the Aurora theme
- **Nyxora Pixel Dark** — futuristic pixel-art dark theme inspired by Nothing OS & CRT terminals
- **Nyxora Pixel Light** — minimal e-ink style light variant of the Pixel theme
- Comprehensive UI styling: Editor, Terminal, Sidebar, Activity Bar, Status Bar, and more
- Consistent syntax highlighting across JavaScript, TypeScript, Python, Go, and more
